# Web App

Next.js frontend for the guided AI webnovel typography workflow.

## Owns

- Step shell and navigation.
- Genre selection.
- Guest draft state.
- Layout editor UI.
- Style input and resolved-list UI.
- Generation waiting UI.
- Candidate selection.
- Release 2 effect editor and browser export.

## Design Source

Read `../../docs/design/DESIGN.md` before changing user-facing UI.

## Local Run

```bash
npm install
npm run dev:web
```

from the repository root.
