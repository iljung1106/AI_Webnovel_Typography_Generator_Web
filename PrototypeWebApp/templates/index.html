Step 1

제목 입력과 초기 배치 생성

제목을 입력하면 `LayoutModule`이 초기 타이포그래피 배치를 생성합니다.

제목을 입력하고 시작하세요.

레이아웃이 생성되면 다음 단계로 이동할 수 있습니다.

Step 2

글자별 위치, 크기, 각도 편집

각 글자를 직접 드래그해 조정합니다. 본문 드래그는 이동, 우하단 핸들은 크기, 상단 핸들은 회전입니다.

레이아웃이 생성되면 편집기가 활성화됩니다.

Drag letter to move Drag corner to scale Drag top handle to rotate

Selection

Glyph Inspector

글자를 하나 선택하면 정밀 조정 값이 여기 표시됩니다.
선택 글자 가
x: 0 / y: 0 / size: 0 / rot: 0°

Step 3

요소와 스타일 키워드로 프롬프트 생성

요소와 키워드는 비워둘 수 있습니다. 제목과 편집된 흐름을 기준으로 `PromptGenerationModule`이 프롬프트를 생성합니다.

Step 4

샘플 생성과 결과 확인

편집된 배치 이미지와 생성된 프롬프트를 기준으로 `ImageGenerationModule`이 샘플 이미지를 생성합니다.
