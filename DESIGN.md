# DESIGN.md

Canonical design direction for this project lives at:

- `docs/design/DESIGN.md`

Implementation agents must read that file before building or changing user-facing UI.

Current status:

- UX direction is defined.
- Final brand identity is still deferred.
- The app should feel white-first, sharp, practical, precise, and subtly fantastical.
- The user typography result should be the main artifact; the UI should stay quiet and exact.
